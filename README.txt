漫画の発売日を管理できるサイトです。
読み続けている漫画がいつの間にか発売されていることがあったので、それを管理できるサイトがあると便利だと思って作成しました。
使用技術
javaScript、Vue.js、Firebase、楽天ブックス書籍検索API
【使用方法】
･PC版の場合は本のところにマウスを持っていくと右上に☆マークが表示され、☆マークをクリックするとお気に入り登録されます。(タブレット、スマートフォン版は最初から表示されています。)☆マークをもう一度クリックするとお気に入り登録が解除されます。
･左側のお気に入りをクリックするとお気に入り登録した本が表示され、発売日が表示されます。
既に販売中なのは｢販売中｣、発売日が1年未満の場合は｢〇〇日｣、1年以上発売日が先になる場合は｢1年以上｣と表示されます。
･本の画像や値段をクリックすると楽天の本の販売ページに移動します。
･ホームをクリックすると最初に表示された画面に戻ります
･上の検索ボックスから検索すると楽天ブックスのAPIから書籍データを取得し、表示します。検索結果の一番下まで画面をスクロールすると自動で次の追加の情報をロードします。
